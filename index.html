In this presentation, we will show OpenIPSL, a library of power system components that work with Wolframâ€™s System Modeler. This demonstration of a faulted bus includes the translation of both PSS/E and CIM model files into WSM and a discussion of the verification and validation of the translations. Translation prevents the need to recreate the model and can more rapidly solve problems. The model is easier to understand for PSS/E users because it is a one-for-one translation. The Common Grid Model Exchange Specification (CGMES) standardizes model exchange in an IEC specification. This translation tool could be the starting point for many model sources.
